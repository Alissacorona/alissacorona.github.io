Color Scheme

When I think of a weather report website I think about the color blue. I wanted the color scheme to match the logo I made and I think the color theme matches the title of the website. The color scheme of this site plan are going to be the same colors as the website. the colors purple, yellow, and black are going to break up the blue so the website does not get swalled up by the color blue.
